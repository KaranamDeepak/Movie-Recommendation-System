# 🎬 Movie Recommendation System

A content-based Movie Recommendation System built using Machine Learning, TF-IDF Vectorization, and Cosine Similarity.  
The system recommends movies similar to the movie entered by the user.

---

# 🚀 Features

- Movie recommendation based on similarity
- TF-IDF text vectorization
- Cosine similarity algorithm
- Flask backend API
- Interactive frontend UI
- Real-time recommendations

---

# 🛠️ Tech Stack

- Python
- Pandas
- Scikit-learn
- Flask
- HTML
- CSS
- JavaScript
